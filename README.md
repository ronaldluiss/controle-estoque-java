# 📦 Controle de Estoque em Java

Projeto simples desenvolvido durante meus estudos de Java no curso do professor Nélio Alves.

O programa permite cadastrar um produto e atualizar a quantidade em estoque através de operações de entrada e saída feitas pelo usuário no console.

---

## 🚀 Funcionalidades

* Informar nome, preço e quantidade do produto
* Calcular o valor total em estoque
* Adicionar produtos ao estoque
* Remover produtos do estoque
* Mostrar os dados atualizados do produto

---

## 🛠️ Tecnologias

* Java
* Programação Orientada a Objetos (POO)

---

## 📚 O que pratiquei neste projeto

* Criação de classes e objetos
* Métodos em Java
* Organização em pacotes
* Uso do Scanner para entrada de dados
* Sobrescrita do método `toString()`

---

```
git clone https://github.com/ronaldluiss/controle-estoque-java.git
```
